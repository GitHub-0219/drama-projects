## 批量生产计划

### 短剧剧本
- 每批产出：10集/部
- 每集结构：开场(10s) + 主体(90s) + 结尾悬念(20s)
- 字数：约500-800字/集

### 小说
- 每章约2500字
- 可同时推进多部
- 需要人物设定确认后再动笔
