## 小说一：《万古第一神》（玄幻仙侠）

### 基本信息
- **类型**：东方玄幻 / 仙侠修真
- **字数**：2000万字（约8000章，每章2500字）
- **核心卖点**：废柴逆袭 + 热血战斗 + 后宫养成 + 宏大世界观
- **目标受众**：男频，18-35岁
- **短视频改编方向**：每章可改编为1-2集短视频，战斗场面为高光片段

### 世界观设定
- **天地结构**：九天十地，三十三重天外天
  - 凡人界（九大洲）
  - 修真界（五大仙域）
  - 仙界（三十三天）
  - 神界（混沌之外）
  - 魔界（深渊之下）
- **修炼体系**：
  - 炼气 → 筑基 → 金丹 → 元婴 → 化神 → 渡劫 → 大乘 → 飞升 → 仙人 → 仙王 → 仙帝 → 神王 → 神帝 → 至高神
  - 每个大境界分九重
- **力量元素**：金木水火土 + 风雷冰暗光 + 混沌元力
- **特殊体系**：炼丹、炼器、阵法、符箓、驭兽、剑道、体修

### 核心人物设定（20人）

**1. 叶尘（男主角）**
- **年龄**：16岁出场 → 最终万岁
- **外貌**：黑发披肩，剑眉星目，身形修长但初期瘦弱。左眼深处有金色光芒（混沌之眼觉醒后）。穿洗得发白的灰色布衣，腰间别一把破旧铁剑。
- **性格**：表面沉默寡言，内心极度坚韧。受辱时不怒，得势时不狂。对敌人冷酷无情，对亲近之人温柔如水。有极强的底线意识——不杀无辜，不欺弱小。
- **背景**：叶家废物三少爷，灵根被夺，沦为笑柄。实为远古混沌神帝转世，体内封印着毁天灭地的力量。
- **成长线**：废物→天才→一方霸主→仙界传奇→至高神帝
- **口头禅**："我叶尘，从不需要别人看得起。"
- **标志性动作**：战斗前闭眼深呼吸，睁眼时瞳孔变成金色
- **短视频名场面**：
  - 灵根被夺时的隐忍
  - 第一次逆袭打脸
  - 金瞳觉醒瞬间
  - 以一敌百的战斗

**2. 苏灵儿（女主角）**
- **年龄**：15岁出场
- **外貌**：一头银白长发如瀑布垂至腰际，肤若凝脂，眸如星辰。常穿淡蓝色仙裙，腰系玉佩，足踏白靴。眉心有一点朱砂痣（冰凤血脉标记）。
- **性格**：外表冷若冰霜，实则心地善良。极度聪慧，精通阵法和炼丹。对叶尘从冷漠到好奇到深爱，感情线缓慢而动人。
- **背景**：北荒苏家嫡女，体内封印着远古冰凤血脉。家族被灭，独自逃亡。
- **成长线**：落难千金→阵法天才→冰凤觉醒→仙界女帝→与叶尘并肩
- **口头禅**："我不需要保护，但我愿意与你并肩。"
- **短视频名场面**：
  - 冰凤血脉觉醒
  - 与叶尘的初遇（互相看不顺眼）
  - 生死关头的告白
  - 冰凤化形战斗

**3. 萧炎（反派/亦敌亦友）**
- **年龄**：18岁出场
- **外貌**：红发如火，赤瞳，身材魁梧，总穿黑色战甲。左脸有一道从眉角到下巴的疤痕。
- **性格**：狂傲不羁，但有自己的道义。前期是叶尘的死敌，后期成为最可靠的战友。信奉"强者为尊"。
- **背景**：南疆萧家天才，被选为火神传承者。与叶尘的矛盾源于两人注定争夺同一个机缘。
- **口头禅**："这天下，只有我萧炎不想争的，没有我争不到的！"
- **短视频名场面**：
  - 与叶尘的第一次对决
  - 火神传承的痛苦蜕变
  - 背叛与救赎的转折
  - 最终并肩作战

**4. 老酒鬼 / 醉仙人（师父）**
- **年龄**：外表60岁，实际数万岁
- **外貌**：邋遢老头，酒糟鼻，总穿破烂灰色道袍，腰间挂个酒葫芦。看似疯疯癫癫，眼神偶尔流露深邃光芒。
- **性格**：表面不正经、贪酒、好色，实则是隐世大能。教导叶尘的方式是"不教"——让他自己悟。
- **背景**：上古仙界大战的幸存者，曾是仙帝级别，因重伤隐居凡人界。
- **口头禅**："小子，天下的道理就一个字——活。活着就是赢。"
- **短视频名场面**：
  - 第一次出场的邋遢形象
  - 随手一招秒杀强敌的反转
  - 传授绝学时的正经时刻
  - 为保护叶尘牺牲的催泪

**5. 叶天雄（叶家家主/叶尘之父）**
- **外貌**：中年男子，方脸浓眉，虎背熊腰。穿锦衣华服，气势威严。
- **性格**：老谋深算，重家族利益。对叶尘表面冷漠，实则有苦衷。
- **背景**：为保护叶尘封印其灵根，却因此被叶尘误解多年。
- **短视频名场面**：父子和解的催泪戏

### 基本信息
- **类型**：都市重生 / 修仙经营

- **核心卖点**：重生逆袭 + 都市装逼 + 经营帝国 + 修仙元素
- **目标受众**：男频，20-40岁

### 核心人物设定（15人）

**1. 陈默（男主角）**
- **年龄**：重生回18岁
- **外貌**：普通偏上的长相，但眼神深邃得不像18岁。戴一副廉价黑框眼镜（后期摘掉）。穿地摊货但气质出众。
- **性格**：前世窝囊悔恨，重生后沉稳果断。表面低调，关键时刻锋芒毕露。对亲人极度保护，对敌人不留情面。

  - 前世记忆闪回的痛苦

- **年龄**：18岁
- **外貌**：清秀佳人，扎马尾，穿校服也好看。不化妆但皮肤极好。笑起来有酒窝。

- **短视频名场面**：雨中等陈默的告白

**3. 赵天龙（反派）**

- **性格**：富二代，嚣张跋扈，视女人为玩物。
- **背景**：前世抢走陈默女友，今生要被陈默碾压。
- **短视频名场面**：被打脸的经典场面

**4. 老药（戒指中的修仙导师）**

## 小说三：《凤临天下》（宫斗/女强）

### 基本信息
- **类型**：古代宫斗 / 女强重生
- **字数**：2000万字
- **核心卖点**：女主重生复仇 + 宫斗权谋 + 女强成长 + 绝美爱情
- **目标受众**：女频，16-35岁

### 核心人物设定（15人）

**1. 沈清歌（女主角）**
- **年龄**：重生回15岁
- **外貌**：鹅蛋脸，柳叶眉，杏眼含情。肤白如玉，唇不点而朱。初期穿素色衣裙，后期凤袍加身。身高165cm，体态纤细但不柔弱。
- **性格**：前世天真善良被人利用，重生后外柔内刚。表面温婉大方，实则心有韬略。对敌人冷酷，对忠善之人温和。
- **背景**：前世是丞相府嫡女，被继母和庶妹联手害死。重生回入宫选秀前一年。
- **成长线**：弱女→宫女→贵妃→皇后→女帝
- **口头禅**："这一世，欠我的，一个都跑不掉。"
- **标志性动作**：微笑时手指轻轻转动手腕上的玉镯（思考/算计的信号）
- **短视频名场面**：
  - 重生后第一次对镜冷笑
  - 在选秀中惊艳全场
  - 反杀继母的权谋戏
  - 登基为帝的凤袍加身

**2. 萧衍（男主角/皇帝）**
- **年龄**：22岁
- **外貌**：龙章凤姿，剑眉入鬓，面如冠玉。穿明黄龙袍时威严冷酷，穿便服时温润如玉。
- **性格**：表面冷酷无情的帝王，实则是最孤独的人。对沈清歌从利用到深爱。
- **背景**：少年登基，被太后和权臣架空。需要一个帮他夺回权力的女人。
- **口头禅**："朕的天下，不需要第二个人来分。"（后期）"除了你。"

**3. 沈清婉（反派/庶妹）**
- **外貌**：妩媚妖艳，善用妆容。总穿艳色衣裙。
- **性格**：表面楚楚可怜，实则心狠手辣。嫉妒心极强。
- **短视频名场面**：被揭穿真面目的崩溃

**4. 太后（大反派）**
- **外貌**：保养极好的中年美妇，凤冠华服。
- **性格**：权力欲极强，心狠手辣但有底线。
- **短视频名场面**：与沈清歌的婆媳对决

## 通用设定说明

### 短视频改编规则
1. **每章 = 1-2集短视频**（2分钟/集）
2. **每集必须包含**：
   - 开头钩子（前5秒抓住观众）
   - 核心冲突/爽点
   - 结尾悬念（让人想看下一集）
3. **高光场面优先改编**：战斗、告白、打脸、反转
4. **人物出场要有辨识度**：固定的出场BGM、标志性动作

### 更新计划
- 每部小说每天更新3章（约7500字）
- 每部约需7年完成（2000万字 ÷ 7500字/天 ÷ 365天）
- 可多部并行更新
