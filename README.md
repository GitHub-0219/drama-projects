# 🎬 短剧与小说企划总库

![Projects](https://img.shields.io/badge/短剧-300部-orange)
![Novels](https://img.shields.io/badge/小说-5部-purple)
![Mythology](https://img.shields.io/badge/题材-神话%2F宠物%2F搞笑-red)

> 📜 中国神话/宠物/搞笑短剧 300 部 + 5 部千万字级小说的总企划库，涵盖世界观、角色设定、剧情大纲等核心文档。

---

## ✨ 功能特性

| 类别 | 说明 |
|------|------|
| 🏛️ 神话短剧 | 女娲补天、后羿射日、精卫填海等经典改编 |
| 🐱 宠物短剧 | 猫和黄瓜的战争、汪星人联盟等萌宠题材 |
| 😂 搞笑短剧 | 生活情景喜剧、反转短剧等 |
| 🐾 萌宠短剧 | 中华田园犬、兔子、仓鼠等萌宠题材 |
| 📖 长篇小说 | 5 部千万字级小说完整企划 |
| 🌍 世界观构建 | 每部作品完整的设定文档 |
| 👥 角色档案 | 详细的人物设定与关系图谱 |

---

## 🛠️ 技术栈

| 技术 | 用途 |
|------|------|
| Markdown | 剧本与企划文档撰写 |
| JSON | 角色数据与剧情结构化存储 |
| Mermaid | 剧情流程图与人物关系图 |
| Excel | 排期与数据管理 |

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/GitHub-0219/drama-projects.git
cd drama-projects

# 浏览神话短剧企划
ls drama-projects/mythology/

# 浏览宠物搞笑短剧企划
ls drama-projects/pet-comedy/

# 浏览萌宠短剧企划
ls drama-projects/pet/

# 浏览长篇小说企划
ls drama-projects/novels/
```
